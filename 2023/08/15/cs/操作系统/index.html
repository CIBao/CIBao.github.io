# 2、线程与进程的比较

更轻、启动快、开销小

共享进程的资源、但有自己的栈和 R

# 3、一个进程可以创建多少线程，和什么有关？

32 位系统下、用户空间 3GB、栈 10M，300 个左右

太多线程切换开销大

# 4、外中断和异常有什么区别？

CPU 执行指令以外的事件（IO 完成终端）

CPU 执行指令内部的事件（除以 0）

# 7、Linux 下进程间通信方式？

管道：无名管道（单向流动、父子进程）有名管道（FIFO，任意进程）

共享内存：映射一段所有进程都能访问的内存

消息队列：内核中具有消息的链表

socket：多机通信

信号：通知进程某个事件发生

信号量：计数器，锁机制用在共享资源访问

# 8、Linux 下同步机制？

信号量：线程 + 进程

互斥锁 + 条件变量：线程

# 9、如果系统中具有快表后，那么地址的转换过程变成什么样了？

如果找到就不用从内存再次访问页表，直接从快找对对应的物理页号，形成物理地址，然后访问

# 10、内存交换和覆盖有什么区别？

交换：不同进程

覆盖：同一进程

# 46、交换空间与虚拟内存的关系

虚拟内存：硬盘的空间

# 47、抖动你知道是什么吗？它也叫颠簸现象

原因：进程的物理块不够
