1、 在 main 执行之前和之后执行的代码可能是什么？

执行前：

设置栈指针，
初始化静态/全局变量（构造函数）
argc, argv参数传递

执行后：

全局对象（析构函数）

2、结构体内存对齐问题？

成员按顺序存储，第一个成员地址是对象首地址
按最大成员 size 对齐

alignas：按指定方式对齐
alignof：返回对齐方式

3. 指针和引用的区别

变量 / 别名，层次，可为空，可改变，大小

指针：变量，多级，空 ，可改变 ，大小不固定
引用：别名，一级，不可空，不可改变，4 字节指针

4、在传递函数参数时，什么时候该使用指针，什么时候该使用引用呢？

指针：返回局内变量的内存
引用：不用创建临时变量，类对象

5、堆和栈的区别

分配方式、速度原理、碎片、大小

栈：系统分配，更快（OS 支持，专门指令，专门寄存器），无碎片，小（4M）
堆：主动申请，更慢（C/C++ 库函数，找合适内存，访问两次），有碎片，大（3G）

8、new /delete 与 malloc /free 的异同

运算符 / 库。主动 / 自动。类型安全。返回指针，new 原理

同：内存申请和释放
异：
new: C++ 运算符，自动计算大小，类型安全，返回对象指针
malloc: C 库函数，手动计算大小，不安全，返回 void 指针

new 本质：封装了 malloc，operator new 运算符（分配大内存），并调用对象构造函数

12、被 free 回收的内存是立即返还给操作系统吗？

链表、系统调用

不是，先存到 free 链表（也会合并内存），下次申请从 free 链表从查找，避免频繁系统调用

13、宏定义和函数有何区别？

原理、时期、返回值、快慢、类型检查

宏定义：无返回值，预处理时文本替换（插入代码）更快。无类型检查
函数： 有返回值， 函数调用需要执行跳转指令。有类型检查
typedef：别名，编译期
const：有分配内存，空间编译期，有类型检查

15、变量声明和定义区别？

分配、次数

声明：不分配内存，多次声明
定义：分配内存，一次定义

16、strlen 和 sizeof 区别？

strlen：库函数，运行，字符指针（'\0'）,
sizeof：运算符，编译，任意类型，不能用在动态分配的对象
