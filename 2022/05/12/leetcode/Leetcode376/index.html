# 题目

https://leetcode.cn/problems/wiggle-subsequence

# 思路

# 方法一：动态规划 O (N^2)

  • up[i] 表示第 i 个为峰的最长序列长度， down[i] 表示第 i 个为谷的最长序列长度
  • up[i] = max(down[j] + 1) where j in [1..i])
  • down[i] = max(up[j] + 1) where j in [1..i])

# 方法二：贪心 O (N)

  • flag==1 代表朝上， flag == 0 代表朝下
  • 如果新的数大于 pre ，那么取走，并更新 flag ， pre
  • 如果不是则让 pre 更加小或更加大，方便下一个数有更大概率被取走
